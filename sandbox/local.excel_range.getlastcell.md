# Excel\_Range.getLastCell method

Gets the last cell within the range. For example, the last cell of "B2:D5" is "D5". 

 \[Api set: ExcelApi 1.1\]

**Signature:**
```javascript
getLastCell(): Excel.Range;
```
**Returns:** `Excel.Range`
