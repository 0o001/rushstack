# Word\_Body.parentBody property

Gets the parent body of the body. For example, a table cell body's parent body could be a header. Throws if there isn't a parent body. Read-only. 

 \[Api set: WordApi 1.3\]

**Signature:**
```javascript
parentBody: Word.Body
```
