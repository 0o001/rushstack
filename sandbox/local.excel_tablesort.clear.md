# Excel\_TableSort.clear method

Clears the sorting that is currently on the table. While this doesn't modify the table's ordering, it clears the state of the header buttons. 

 \[Api set: ExcelApi 1.2\]

**Signature:**
```javascript
clear(): void;
```
**Returns:** `void`
