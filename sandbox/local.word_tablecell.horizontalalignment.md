# Word\_TableCell.horizontalAlignment property

Gets and sets the horizontal alignment of the cell. The value can be 'left', 'centered', 'right', or 'justified'. 

 \[Api set: WordApi 1.3\]

**Signature:**
```javascript
horizontalAlignment: string
```
