# Word\_ContentControl.delete method

Deletes the content control and its content. If keepContent is set to true, the content is not deleted. 

 \[Api set: WordApi 1.1\]

**Signature:**
```javascript
delete(keepContent: boolean): void;
```
**Returns:** `void`
