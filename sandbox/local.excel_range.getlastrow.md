# Excel\_Range.getLastRow method

Gets the last row within the range. For example, the last row of "B2:D5" is "B5:D5". 

 \[Api set: ExcelApi 1.1\]

**Signature:**
```javascript
getLastRow(): Excel.Range;
```
**Returns:** `Excel.Range`
