# Word\_TableRow.verticalAlignment property

Gets and sets the vertical alignment of the cells in the row. The value can be 'top', 'center' or 'bottom'. 

 \[Api set: WordApi 1.3\]

**Signature:**
```javascript
verticalAlignment: string
```
