# Word\_TableRow.isHeader property

Checks whether the row is a header row. Read-only. To set the number of header rows, use HeaderRowCount on the Table object. 

 \[Api set: WordApi 1.3\]

**Signature:**
```javascript
isHeader: boolean
```
