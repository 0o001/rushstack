# Word\_Table.horizontalAlignment property

Gets and sets the horizontal alignment of every cell in the table. The value can be 'left', 'centered', 'right', or 'justified'. 

 \[Api set: WordApi 1.3\]

**Signature:**
```javascript
horizontalAlignment: string
```
