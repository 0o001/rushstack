# Excel\_Range.cellCount property

Number of cells in the range. This API will return -1 if the cell count exceeds 2^31-1 (2,147,483,647). Read-only. 

 \[Api set: ExcelApi 1.1\]

**Signature:**
```javascript
cellCount: number
```
