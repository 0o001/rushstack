# Word\_Font.strikeThrough property

Gets or sets a value that indicates whether the font has a strike through. True if the font is formatted as strikethrough text, otherwise, false. 

 \[Api set: WordApi 1.1\]

**Signature:**
```javascript
strikeThrough: boolean
```
