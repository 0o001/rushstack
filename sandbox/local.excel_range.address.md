# Excel\_Range.address property

Represents the range reference in A1-style. Address value will contain the Sheet reference (e.g. Sheet1!A1:B4). Read-only. 

 \[Api set: ExcelApi 1.1\]

**Signature:**
```javascript
address: string
```
