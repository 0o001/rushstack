# Word\_Table.verticalAlignment property

Gets and sets the vertical alignment of every cell in the table. The value can be 'top', 'center' or 'bottom'. 

 \[Api set: WordApi 1.3\]

**Signature:**
```javascript
verticalAlignment: string
```
