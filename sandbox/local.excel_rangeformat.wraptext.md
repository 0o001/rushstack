# Excel\_RangeFormat.wrapText property

Indicates if Excel wraps the text in the object. A null value indicates that the entire range doesn't have uniform wrap setting 

 \[Api set: ExcelApi 1.1\]

**Signature:**
```javascript
wrapText: boolean
```
