# Word\_Paragraph.lineSpacing property

Gets or sets the line spacing, in points, for the specified paragraph. In the Word UI, this value is divided by 12. 

 \[Api set: WordApi 1.1\]

**Signature:**
```javascript
lineSpacing: number
```
