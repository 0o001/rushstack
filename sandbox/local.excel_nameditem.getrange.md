# Excel\_NamedItem.getRange method

Returns the range object that is associated with the name. Throws an error if the named item's type is not a range. 

 \[Api set: ExcelApi 1.1\]

**Signature:**
```javascript
getRange(): Excel.Range;
```
**Returns:** `Excel.Range`
